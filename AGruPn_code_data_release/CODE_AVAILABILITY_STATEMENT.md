# Code Availability Statement

The authors provide the main public code and data for the AGruPn study in this release. The repository contains released CSV data, partial implementation notebooks, metric scripts, figure-generation scripts, and a public training-protocol file.

The complete integrated AGruPn core training code, unpublished symbolic-physics components, trained model checkpoints, and raw private data are not included because they are part of ongoing follow-up research. The released materials are sufficient to inspect the data protocol, reproduce the public evaluation metrics from the provided prediction sequences, and regenerate the public analysis figures.
